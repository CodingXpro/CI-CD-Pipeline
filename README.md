		Smart Bookmark App

Build and deploy the simple bookmark manager with the following requirements

1.user can signup and login using google (no email/password-google auth only)

2.A looged in user can add bookmark(url + title)

3.Bookmarks are private to each user (userA can not see the userB's bookmarks)

4.Bookmarks list updates in real time without page refresh (if you open two tabs and add a bookmark in one ,it should appear in another)

5.user can delete their own bookmarks.

6. App must be deployed on vercel with the working url.

Tech Stack- Next.js(App router not pages router),supabase(auth,database,real time),Tailwind Css for basic styling.
